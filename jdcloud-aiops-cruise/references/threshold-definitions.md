# 阈值定义 & 实例规格上限速查

> 所有阈值是默认推荐值，可通过 `--threshold-override` 参数覆盖。

## 通用阈值

| 指标 | Warning | Critical | 说明 |
|---|---|---|---|
| CPU 使用率 % | > 70 | > 85 | 超过 85% 可能触发限频 |
| 内存使用率 % | > 80 | > 90 | 超过 90% OOM 风险高 |
| 磁盘使用率 % | > 75 | > 90 | 超过 90% 应立即扩容 |
| 系统负载 / vCPU 数 | > 4.0 | > 6.0 | 超过 6.0 表示严重过载 |
| 环比增长率 % | > 30% | > 50% | 与昨日同期对比 |

### MongoDB

| 指标 | Info | Warning | Critical | 说明 |
|---|---:|---:|---:|---|
| CPU 使用率 | ≥ 60% | ≥ 70% | ≥ 85% | 高 CPU 需关联慢查询/QPS/IOPS |
| 内存使用率 | ≥ 70% | ≥ 75% | ≥ 90% | 判断 working set 是否超过内存 |
| 磁盘使用率 | ≥ 70% | ≥ 80% | ≥ 90% | 超过 90% 应立即扩容或清理 |
| 连接使用率 | ≥ 60% | ≥ 70% | ≥ 85% | QPS 未同步上升时优先怀疑连接泄漏 |
| 读/写/命令延迟 | ≥ 50ms | ≥ 100ms | ≥ 500ms | 需结合慢查询和 IOPS |
| 复制延迟 | ≥ 5s | ≥ 10s | ≥ 60s | 持续 3 个采样点更可信 |
| Oplog window | ≤ 48h | ≤ 24h | ≤ 6h | 恢复窗口不足风险 |
| 磁盘 7d 增长 | ≥ 5pp | ≥ 10pp | ≥ 20pp | 用于容量预测 |

### CLB

| 规格 | 最大并发连接 | 最大新建连接/秒 | 带宽(Gbps) | Info 阈值 | Warning 阈值 |
|---|---:|---:|---:|---:|---:|
| 标准型 | 500,000 | 50,000 | 5 | ≥ 60% | ≥ 80% |
| 高性能型 | 2,000,000 | 100,000 | 10 | ≥ 60% | ≥ 80% |

CLB 升级评估只生成建议：超过 Warning 阈值时，报告应提示人工通过 `jdcloud-clb-ops` 评估升配/升级；健康后端数 `< 2` 先按 Critical 修复后端健康。

### EIP

| 指标 | Info | Warning | 说明 |
|---|---:|---:|---|
| 入带宽峰值 / EIP带宽 | ≥ 50% | ≥ 80% | 高位持续时建议人工评估升带宽 |
| 出带宽峰值 / EIP带宽 | ≥ 50% | ≥ 80% | 高位持续时建议人工评估升带宽 |
| 绑定状态 | 已绑定记录审计信息 | 未绑定 | 未绑定只提示可能闲置，释放必须委托 `jdcloud-eip-ops` 并人工确认 |

## 安全规则阈值

| 规则 | 级别 | 判定 |
|---|---|---|
| CIDR = `0.0.0.0/0` AND 端口 = 22/3389/5432/3306/6379 | Critical | 管理/数据库端口对公网开放 |
| CIDR = `0.0.0.0/0` AND 端口 > 1024 | Warning | 大范围端口对公网开放 |
| 安全组规则数 > 50 | Warning | 规则过多，建议优化 |

## 安全合规阈值

| 检查项 | 级别 | 判定 |
|---|---|---|
| 云硬盘未加密 (数据盘) | Warning | 敏感数据建议开启加密 |
| 云硬盘未加密 (系统盘) | Info | 敏感环境建议开启 |
| Redis 淘汰策略 = noeviction | Warning | 内存满时写入会失败 |
| Redis 版本 < 5.0 | Info | 建议升级到5.0+ |
| MongoDB 端口 27017 对 `0.0.0.0/0` 开放 | Critical | 数据库端口公网暴露 |
| MongoDB 复制延迟 > 60s | Critical | 存在读一致性和故障切换风险 |
| MongoDB 磁盘使用率 > 90% | Critical | 写入失败和备份失败风险 |
